## 🛠️ 가계부 연동 가이드

### 1. 구글 스프레드시트 데이터 유효성 검사 (드롭다운)
* 월별 시트의 **25행** `F25`(분류) 셀에 설정할 드롭다운 항목 값 목록:
  `생활비, 교통/차량, 주거/통신, 보험, 의료/건강, 여행/여가, 육아, 용돈, 기타, 투자/저축, 수입, 가족`

### 2. 가계부 자동 파이프라인 (Apps Script) 자동 분류
* 드라이브에 PDF 명세서, 현대카드 이용내역 HTML, 이체 확인 스크린샷 이미지 등이 수신되면, Gemini API가 위 키워드 테이블 및 분류 범례를 참조하여 **12대 카테고리로 자동 정리**해 시트에 yellow 백그라운드로 자동 업로드하게 됩니다.
